# 介绍

这是关于组件CreateNameRule的相关说明，路径在src/components/CreateNameRule.vue。组件CreateNameRule用于创建命名规则。

# API

# CreateNameRule props

属性 说明 类型 默认值
value 完全只读,包含输入的值以及分隔符 Object -
module 完全只读，指所在的模块，如tiktok、gdt... String -
availableFields 完全只读，指可使用的字段 Array -
templateType 完全只读，指当前所在的模板，如ad、adset、campaign String -
draft 完全只读 Object 空
accountId 完全只读，指当前广告账号的id String -
error 完全只读，返回的错误信息 String 空
label 完全只读 String 空
labelWidth 完全只读 Number -
hidePreview 完全只读 Boolean -
previewMethod 完全只读，对accountId, type等参数进行处理 Function -
tipLabel 完全只读 String -
labelPosition 完全只读 String -
NAMING_RULE_SEPARATOR 命名规则的分隔符 String -
namingRuleList Array 空
selectedRuleId String 空
nameRule 存储着分隔符，以及进行命名的值 Object -
namingRulePreviewName String 空
